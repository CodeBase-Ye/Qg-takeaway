# 秋果外卖
&ensp;&ensp;&ensp;项目是专门为餐饮企业定制的一款软件产品，包括系统管理后台和移动端应用两部分。其中系统管理后台主要提供给餐饮企业
内部员工使用，可以对餐厅的分类、菜品、套餐、订单、员工等进行管理维护。移动端应用主要提供给消费者使用，可以在线
浏览菜品、添加购物车、下单等。
